Buttons trigger actions. SEMANTICA provides two variants — primary and outline.

## Usage
```html
<a href="#" class="btn-primary">Get Started →</a>
<a href="#" class="btn-outline">Watch Demo</a>
```

## CSS classes

| Class | Description |
|-------|-------------|
| `.btn-primary` | Filled orange button |
| `.btn-outline` | Transparent with border |
